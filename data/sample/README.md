# Sample Data

이 폴더의 `ceo_event_returns.csv`는 baseline 계산을 검증하기 위해 만든 합성 데이터입니다.

## Columns

- `event_id`
- `event_date`
- `symbol`
- `event_tone`
- `market_return`
- `stock_return_t_minus_1`
- `stock_return_t0`
- `stock_return_t_plus_1`

## Usage

```bash
python -m src.run_baseline
```

강의자료, 과제 원본 데이터, 유료 데이터는 포함하지 않습니다.
